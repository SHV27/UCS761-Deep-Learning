# Research Notes

Project Title:
Semi-Supervised Transfer Learning for Galaxy Morphology Classification

Objective:
To classify galaxy morphologies (spiral, elliptical, irregular) using deep learning with limited labeled data.

Approach:
Use a pretrained CNN (ResNet/DenseNet) through transfer learning and apply pseudo-labeling to utilize unlabeled astronomical survey images.

Dataset:
Galaxy Zoo 2 and DECaLS astronomical survey images.

Evaluation Metrics:
Accuracy, Macro-F1 score, Precision, Recall and Confusion Matrix.

Next Steps:
Prepare dataset pipeline and implement baseline CNN model.
